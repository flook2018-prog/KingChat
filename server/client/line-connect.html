🔗 เชื่อมต่อ LINE Official Account

เชื่อมต่อบัญชี LINE OA ของคุณเข้ากับระบบ KingChat

1 สร้างบัญชี LINE Developers

เข้าไปที่ LINE Developers Console และสร้างแอปพลิเคชันใหม่:

  • ไปที่ developers.line.biz/console
  • สร้าง Provider ใหม่หรือเลือก Provider ที่มีอยู่
  • คลิก "Create a new channel" → เลือก "Messaging API"
  • กรอกข้อมูลและยอมรับเงื่อนไข
2 ตั้งค่า Webhook URL

คัดลอก Webhook URL นี้ไปใส่ในหน้า LINE Developers Console:

🌐 Webhook URL ของคุณ:
https://kingchat.up.railway.app/webhook/line

ขั้นตอน:

  • ไปที่แท็บ "Messaging API" ใน LINE Console
  • เลื่อนลงไปหาส่วน "Webhook settings"
  • ใส่ URL ด้านบนและเปิดใช้งาน "Use webhook"
  • คลิก "Verify" เพื่อทดสอบการเชื่อมต่อ
3 ดึงข้อมูลการเชื่อมต่อ

คัดลอกข้อมูลเหล่านี้จาก LINE Developers Console:

📍 ตำแหน่งข้อมูลใน Console:

  • Channel ID: หน้า "Basic settings" → Channel ID
  • Channel Secret: หน้า "Basic settings" → Channel secret (คลิก Issue)
  • Channel Access Token: หน้า "Messaging API" → Channel access token (คลิก Issue)

🔑 กรอกข้อมูลการเชื่อมต่อ

ชื่อที่จะแสดงในระบบ (ตั้งเองได้)
ตัวเลข Channel ID จาก LINE Developers Console
Channel Secret จากหน้า Basic settings
Channel Access Token จากหน้า Messaging API
